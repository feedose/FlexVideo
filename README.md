# FlexVideo

**Превратите любой текст в готовое видео за несколько кликов.**

FlexVideo — программа для автоматического создания видеороликов на Windows и macOS. Вставьте сценарий, выберите режим и нажмите «Старт». Программа разделит текст на сцены, создаст изображения или видео, добавит озвучку, субтитры, музыку и соберёт готовый ролик.

FlexVideo подходит для длинных видео YouTube, Rutube, TikTok, информационных каналов, историй, обзоров и других форматов. Можно создавать ролики с помощью нейросетей или собирать их из собственных фотографий и видео.

Возможности FlexVideo:

- генерация изображений и видео по тексту;
- автоматическая озвучка разными голосами;
- создание и оформление субтитров;
- фоновая музыка с автоматическим приглушением;
- плавный зум, движение кадров и визуальные эффекты;
- сохранение внешности персонажей между сценами;
- просмотр и перегенерация отдельных фрагментов;
- создание заголовков, описаний, тегов и превью;
- сохранение разных настроек в отдельных профилях.

Все проекты сохраняются на вашем компьютере. Вам не нужно разбираться в монтаже или сложных программах: FlexVideo выполняет основную работу автоматически и позволяет быстро получать готовые ролики для публикации.
